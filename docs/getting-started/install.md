# 1. Install the app

Schuly runs on **Android** and **iOS**.

- **Get it from [schuly.dev](https://schuly.dev)** - the website always links to the
  latest builds for each platform.
- **Android** - install from the store listing, or sideload the release `.apk` from the
  [GitHub releases](https://github.com/schulydev/Schuly/releases). If you sideload,
  Android will ask you to allow installs from your browser/files app the first time.
- **iOS** - install from the store listing linked on the website.

:::info Two flavors
You may see **Schuly** and **Schuly DEV**. Install **Schuly** - "DEV" is a separate
test build used by contributors and can run side by side without touching your real data.
:::

Once it's installed, open it and continue to
**[choosing how you sign in](choose-a-mode.md)**.
